# PerforatedAI-Community-Projects
This repo is to for folks who are building with Perforated AI to collaborate.  Please create forks and and PRs in here.  This will be significantly less moderated than the official examples folders of our main repo.  Main repo will be reserved for braoder use cases and proof of cocnepts for new verticals.  This repo is ideal for:
- Collaborating on Inception for specific hackathons
- Sharing your tranfer learning examples on custom datasets
- Working together on projects where you were unable to get a successful outcome but would like to share a starting point for others to continue

## Challenge Projects
